# Contact Form with JavaScript Validation

A modern, responsive contact form with real-time validation and smooth animations.

## Features

- **Real-time Validation** - Instant feedback as users type
- **Visual Feedback** - Color-coded borders and icons
- **Responsive Design** - Works on all device sizes
- **Smooth Animations** - Fade-in, slide-down, and shake effects
- **Modern UI** - Gradient design with clean aesthetics

## Validation Rules

| Field | Validation | Error Message |
|-------|------------|---------------|
| Name | Required | "Please enter your full name" |
| Email | Required + Valid format | "Please enter a valid email address" |
| Message | Required + Min 10 chars | "Message must be at least 10 characters long" |

## Technologies Used

- HTML5
- CSS3 (Flexbox, Gradients, Animations)
- JavaScript (ES6)
- Font Awesome Icons

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge
